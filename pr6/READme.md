За семестр были выполнены пять практических работ.

В первой работе сверстали несколько страниц с использованием HTML и SCSS.

Во второй работе изучили основы JavaScript и написали несколько скриптов для работы с DOM.

В третьей работе создали простое React-приложение с компонентами и состоянием.

В четвертой работе написали сервер на Express с CRUD-операциями для управления пользователями.

В пятой работе доработали сервер из четвертой работы, добавили логирование запросов и подключили Swagger для автоматической генерации документации. В результате получили работающее приложение с бэкендом на Express и интерактивной документацией по адресу localhost:3000/api-docs.
